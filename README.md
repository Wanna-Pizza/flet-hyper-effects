# flet-hyper-effects
FletHyperEffects control for Flet

## Installation

Add dependency to `pyproject.toml` of your Flet app:

* **Git dependency**

Link to git repository:

```
dependencies = [
  "flet-hyper-effects @ git+https://github.com/MyGithubAccount/flet-hyper-effects",
  "flet>=0.27.6",
]
```

* **PyPi dependency**  

If the package is published on pypi.org:

```
dependencies = [
  "flet-hyper-effects",
  "flet>=0.27.6",
]
```

Build your app:
```
flet build macos -v
```
